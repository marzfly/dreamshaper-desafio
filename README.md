## A demanda
Criar um sistema que fosse capaz de calcular o salário final de um funcionário e o bônus recebido dependendo da categoria onde ele mais se enquadrasse (padrão, comissionado ou de produção). Seria necessário adicionar os funcionários a um array e exibir a folha de pagamento com as informações de cada funcionário (número de registro, nome completo, bônus e salário final (sabendo que o salário-base equivale a 2000)
## Regras
- Exibir o nome, registro, bonificação e salário final de cada funcionário
- Armazenar as informações de cada colaborador em um ArrayList
- Determinar as funcionalidades do programa (cadastrar cada tipo de funcionário, ver folha de pagamento e sair)
## Processo
- Criar uma constante para o salário base
- Importar as bibliotecas ArrayList e Scanner
- Declarar as variáveis necessárias
- Criar cada array (nome, registro, bônus e salário final)
- Colocar o menu e o switch dentro de um do-while cuja condição de parada é o input do número 0 (sair)
- Solicitar input do usuário e colocar o input dentro do array correspondente
- Calcular o bônus e o salário final
